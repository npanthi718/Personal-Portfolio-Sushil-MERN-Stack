## ✨ Overview

This is a **high-performance, modern, and fully dynamic** Personal Portfolio built with the MERN stack. It features a sophisticated **Admin Dashboard** that allows you to manage every aspect of your professional presence in real-time.

---

## 🚀 Key Features

- **⚡ Real-time Updates:** Powered by Server-Sent Events (SSE) for instant dashboard-to-frontend synchronization.
- **🛠️ Dynamic Admin Panel:** Manage Skills, Projects, Experience, Education, and more without touching the code.
- **📱 Responsive Design:** Flawless experience across Mobile, Tablet, and Desktop devices.
- **📂 File Management:** Integrated with Cloudinary for seamless Resume and Image hosting.
- **🛡️ Secure Auth:** JWT-based authentication with cookie-based security for the Admin Panel.
- **🎨 Modern UI:** Sleek dark-themed interface with teal accents and smooth Framer Motion animations.

---

## 🛠️ Tech Stack

| Frontend | Backend | DevOps |
| :--- | :--- | :--- |
| React 19 & Vite | Node.js & Express | Docker & Docker Compose |
| Material UI (MUI) | MongoDB & Mongoose | Cloudinary (Assets) |
| Framer Motion | JWT & Bcrypt | Vercel / Render |
| CSS Modules | Joi Validation | SSE (Real-time) |

## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/npanthi718/Project-3_Personal-Portfolio.git
cd Project-3_Personal-Portfolio
```

### 2️⃣ Environment Configuration
Create `.env` files in both `frontend` and `backend` directories as per their respective READMEs.

### 3️⃣ Run with Docker (Recommended)
```bash
docker-compose up --build
```

### 4️⃣ Manual Setup
**Backend:**
```bash
cd backend && npm install && npm start
```
**Frontend:**
```bash
cd frontend && npm install && npm run dev
```
