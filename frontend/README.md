## ✨ Features

- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
- **Admin Panel:** Built-in dashboard to manage every section of the portfolio.
- **Dynamic Content:** Home, About, Skills, Projects, Experience, and Education are all fetched from the backend.
- **Real-time Updates:** Instant UI updates using Server-Sent Events (SSE).
- **Interactive Components:** Skill progress bars, Project cards with tech chips, and Contact form.
- **Theme Support:** Dark-themed UI with teal accents for a sleek look.

---

## 🛠️ Tech Stack

- **Framework:** React 19 ⚛️
- **Build Tool:** Vite ⚡
- **UI Components:** Material-UI (MUI) 🎨
- **Styling:** CSS Modules 📦
- **Icons:** React Icons & MUI Icons 💠
- **State Management:** React Context API & Hooks 🔄
- **API Communication:** Fetch API 🌐

## 🚀 Getting Started

### 📋 Prerequisites

- Node.js (v18+)
- Running Backend Server

### ⚙️ Configuration

Create a `.env` file in the `frontend` folder:

```env
VITE_API_URL=http://localhost:5000
```

### 🛠️ Installation

1. **Navigate to the frontend directory:**
   ```bash
   cd frontend
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
